Slot: expression_to_value_mappings

A mapping table in which the keys are boolean expressions and the values are literal results. On enum derivations, used for scalar binning: each key is evaluated with value() bound to the incoming value, and the first truthy key's value is returned as the target permissible value. See issue #99.
